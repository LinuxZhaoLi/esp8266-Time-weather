1.black\white\3d_60是图片库，我只用了外面那几个，大雨中雨还没添加进代码，你可以自己添加。
2.u8g2是OLED的库，自行安装
3.PCLCD2002是取模软件，
4.ArduinoJson\Seniverse\Time这三个包也要安装，第一个不多说，第二个是太极创客写的天气预报的心知天气库，也可以自己JASON获取，第三个是网络授时的包。
5.建议你们改进的地方（我最近没搞这东西了）：
  加入utf-8中文显示，
  将今天的天气转为实时获取的不是那种今天的天气预报（虽然也是实时但调用的JASON不一样，现在更像是预测接下来今天什么天气而不是现在的），
  更多天气图片加进去（取模看下那个天气返回的码就知道了），具体码参看https://docs.seniverse.com/api/start/code.html
  优化显示、过渡动画等
  加入实时调用系统 更精准
